# 📊 Student Marks Prediction

## 📌 Overview

This project predicts students' final exam scores using machine learning techniques based on academic and behavioral data.
It helps in identifying performance trends and supports early academic intervention.

---

## ⚙️ Tech Stack

* Python
* NumPy, Pandas
* Scikit-learn
* XGBoost
* Matplotlib

---

## 🧠 Model Used

* XGBoost Regressor

---

## 📊 Results

* R² Score: ~0.75
* Mean Squared Error: ~10.5

---

## 🎯 Key Idea

The model analyzes factors like study hours, attendance, and past scores to predict final exam performance.

## 🚀 Future Scope

* Hyperparameter tuning
